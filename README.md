# How to download it?
```
git clone https://github.com/Trenek/JTUDZK_Template --recursive
```
Alternatively
```
git clone https://github.com/Trenek/JTUDZK_Template
cd JTUDZK_Template
git submodule update --init --recursive
```
# How to compile it?
```
cmake -B build
cmake --build build --target all -j16
```
